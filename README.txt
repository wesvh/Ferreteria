Instrucciones de uso para el profesor:

Ejecutar el Script SQL que se encuentra en ./SQL.script .
Modificar las credenciales de acceso a la base de datos en funcion de las propias, que se encuentra en ./src/pswd.properties .
Importar la carpeta completa a NetBeans.
Agregar la Libreria de JConnector a Netbeans.
Abrir ./src/View/Menu.java sí se desea ver la interfaz grafica con todo su funcionamiento.
Abrir ./src/Ferreteria/MainTemporal.java sí se desea ver el funcionamiento por consola.
// DATOS EXTRA

Sí se desea ver el funcionamiento de la logica de conexion a la base de datos, se encuentra en ./src/Ferreteria/Conexion.java .
